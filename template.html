Cette revue de presse est écrite en html et donc en utilise les principales balises. De plus, la gestion de la mise en page — dite responsive design — est réalisée par le framework Bootstrap de Twitter. Il est donc important de respecter le système de positionnement (grid design) des différents composants (texte, images, etc.).
Enfin, pour toutes les images, seuls les liens hypertextes sont autorisés. Aucune image ne pourra être installée sur le serveur Github.
